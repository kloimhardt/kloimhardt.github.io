SICM

This is the decription for the clj-tiles chapter at URL https://kloimhardt.github.io/cljtiles.html?page=SICM001

That particular chapter of clj-tiles is called sicm-book-vscheme-part1.

For general information on clj-tiles open https://github.com/kloimhardt/clj-tiles

The chapter shows examples of the MIT open access book "Structure and Interpretation of Classical Mechanics (SICM)" https://mitp-content-server.mit.edu/books/content/sectbyfn/books_pres_0/9579/sicm_edition_2.zip/chapter001.html

In additon to the tutorial of the chapter, there is description of how to begin without any knowledge: https://kloimhardt.github.io/blog/literatur/2023/03/16/competence-comprehesion-2.html

clj-tiles uses the great Emmy Algebra System down

The code of all examples can also be viewed, modified and run on-line within a simple html page https://kloimhardt.github.io/blog/html/sicmutils-as-js-book-part1.html

The code of all examples can also be viewed on the Clojure Civitas site https://clojurecivitas.github.io/mentat_collective/emmy/sicm_ch01.html

There is also an unofficial version of the SICMechanics book with nice rendering https://tgvaughan.github.io/sicm/

Sam Ritchie, the author of Emmy, has a full LaTex source version of the SICM book. https://github.com/mentat-collective/sicm-book

There is also a Jupyter notebook for this chapter https://github.com/kloimhardt/clj-tiles/blob/master/jupyter_notebooks/emmy-sicm-ch01.ipynb

That Jupyter Notebook can be executed with Noj down

An older verion of the Jupyter notebook, using a different visualization, is in https://github.com/sicmutils/sicmutils/blob/main/jupyter/book-examples.ipynb

FDG

This is the decription for the clj-tiles chapter at URL https://kloimhardt.github.io/cljtiles.html?page=FDG001

That particular chapter of clj-tiles is called fdg-book-vscheme-chapter001.

For general information on clj-tiles open https://github.com/kloimhardt/clj-tiles

This is the first chapter of the MIT open-access book "Functional Differential Geometry (FDG)" https://mitpress.mit.edu/9780262019347/functional-differential-geometry/

clj-tiles uses the great Emmy Algebra System down

The code of all examples can also be viewed and executed on the Clojure Civitas site https://clojurecivitas.github.io/mentat_collective/emmy/sicm_ch01.html

The code of all examples can also be viewed and executed on the Maria Cloud site https://2.maria.cloud/gist/65d96869f9a901b243df06f996c1d707

Sam Ritchie, the author of Emmy, has a full LaTex source version of the FDG book. https://github.com/mentat-collective/fdg-book/tree/main/scheme

Emmy

Emmy is a Computer Algebra System https://github.com/mentat-collective/emmy

The official manual for Emmy is https://emmy.mentat.org/

The one who started with the Emmy Algebra System is Colin Smith https://github.com/littleredcomputer

Emmy has been brought to its 2025 form by Sam Ritchie https://samritchie.io

Sam also has a special blog for Emmy https://roadtoreality.substack.com/

Within this blog, Sam describes various talks he gave, including https://sritchie.github.io/clojure-conj-2023/notebooks/conj/quartic_well.html

Somehow similar to the Civitas project (link above) Sam already earlier started an interactive notebook project for Emmy https://reality.mentat.org/
